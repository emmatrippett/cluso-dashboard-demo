Portfolio Demo
Cluso Investor Dashboard
A live, interactive demo of the internal & investor-facing dashboard I built for Cluso during my internship. Everything here is real code — just running on sample data.
My Role
Solo designer & engineer — scoped the product, designed the UX, and shipped every line of code. Worked directly with the CEO to define KPIs investors actually care about, and built three role-based views (Admin / External / Investor) from a single codebase.
What's in here
Financials, revenue, pipeline, marketing, impact, year-on-year comparisons, editable targets, project tracker, role-based access, configurable integrations, and a curated investor view. Everything is interactive — click, edit, customise which metrics you see.
In production
Financial data syncs live from QuickBooks; sales & customer data flows from the team's CRM. Admins can still override or annotate any field. Realtime updates push to every connected viewer via Supabase subscriptions.
Stack
Vanilla JS Chart.js Supabase Postgres Edge Functions Realtime Vercel Resend Gemini API
